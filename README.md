# Coke-Sales-Analysis (Dashboard creation using MS Excel)
## Project Objective
This project focuses on sales performance analysis of beverage products using Microsoft Excel.
The objective is to extract business insights from raw sales data by applying data cleaning, pivot analysis, dashboards, and statistical interpretation.

## The analysis helps identify:
Revenue contribution by brand
Regional performance trends
Sales vs Units Sold comparison  
Operating profit insights
Pricing impact on sales
